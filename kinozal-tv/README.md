## Аниме-индексаторы для трекера КИНОЗАЛ-ТВ


### kinozal-tv-anime.yml (не рекомендуется)

Версия с максимальным количеством сохраняемой информации. Низкая точность, нормированные названия. С большой вероятностью сплит-куры не прочтутся правильно.

### kinozal-tv-anime_(season-packs).yml
Индексатор, ориентированный на Season-Pack-закачку (есть такой фильтр при Ineractive Search). Намного лучше работает со сплит-курами.

Необычный случай, когда парсеру Sonarr лучше дать меньше информации для поиска.

Проводит множество махинаций с оригинальным заголовком:
 - Преобразует "1-12 серии из 12" и подобные им в E01-12
 - Удаляет "E01-12" и подобные им из результата, если сезон завершён (помогает в поиске Season-Pack-ов)
 - Оставляет "E01-12" и подобные им для заголовков, в которых "Part N" для лучшего поиска сплит-куров и для незаконченных релизов для возможности поиска посерийно
 - Удаляет год из раздачи для всех сезонов кроме первого (стандарт парсера Sonarr — год релиза первого сезона, и наличие года самого сезона обзывает результат "Unknown Series")
