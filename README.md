# rutracker-prowlarr-anime-RU

Описание торрент-трекера [rutracker.org](https://rutracker.org/) в формате [Prowlarr Cardigann YML](https://wiki.servarr.com/prowlarr/cardigann-yml-definition), предназначенное исключительно для аниме-разделов. 

Некоторые релизы переводятся TheTVDB довольно вольно, и возможности их найти внутри Sonarr нет. Пример: "Love, Chunibyo & Other Delusions!". На Rutracker только одна раздача имеет это название среди альтернативных, и только по этому названию она совпадает с алиасами TheTVDB. Так на некоторых тайтлах вы получите неполный поиск (вероятнее всего на тех, у которых необычное или вычурное название). На данный момент поменять TheTVDB на что-то более актуальное или практичное нельзя.
